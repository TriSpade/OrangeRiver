Project Orange River

Problem Statement

Intelligent River develops and operates hydrological observation systems to support research and provide real-time monitoring, analysis, and management of water resources. The program has typically focused on resilient data collection. However, these systems are missing the infrastructure to support real-time decision making.

Solution Statement

With this project, we want to create an implementation of cloud-based services that ingest both historical and streaming data collected from the multiple sensors and sources that exist in the Intelligent River systems already. These services will allow for the analytical exploration of the following topics in real-time:

  • Visualization of streaming data
  • Detection of anomalies
  • Event detection or classification
  • Clustering of similar data points for visualization
  • Prediction of future sensor readings or events
